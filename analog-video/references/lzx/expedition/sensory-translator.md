# Sensory Translator (Expedition series)

Source: LZX community Legacy Docs ("All About Sensory Translator").

**Function:** Five-channel **audio envelope follower** — turns sound into five channels of CV. Each channel is a band-passed frequency band whose CV tracks that band's volume. Use the onboard omnidirectional electret mic, an external audio input, or both.

**Controls & connections:**
- **Microphone out** — modular-level signal from the onboard mic.
- **+/- gain switches** — three preamp gain settings; external input selectable between line and modular (±5 V) level.
- **Upper row of outputs** — the individual filtered frequency bands.
- **Bottom row of outputs** — the envelope signal for each band.
- **Amp/Imp switches** — envelope output (Amp) vs trigger/impulse output (Imp, beat-to-trigger).
- **Fast/Slow switches** — envelope response/slew speed.
- Onboard 2-input mixer with Bass/Treble EQ; per-channel LED indicators.

**Filter bands:** five 4-pole bandpass filters, each 2-octave bandwidth, centers at **40, 160, 640, 2560, 10240 Hz** (≈4× spacing — roughly kick / snare / mids / highs / cymbals, with enough isolation between bands).

**Specifications:** Expedition · 16 HP · +12 V 80 mA / −12 V 70 mA.
